# Phase 1: Foundation and Scaffolding

## Objective

Create a clean monorepo base that can host the web app, API service, graph worker, and local infrastructure.

## Deliverables

- Root workspace configuration.
- `apps/web` Next.js skeleton.
- `services/api` TypeScript API skeleton.
- `workers/graph` FastAPI skeleton.
- `infra/docker-compose.yml`.
- `infra/.env.example`.
- README run instructions.

## Exit Criteria

- Project structure matches the build brief.
- Dependencies are declared.
- A developer can see how to start infra, web, API, and worker.
